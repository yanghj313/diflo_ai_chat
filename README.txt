AI Agent Chat Simulation (UI Demo)

이 프로젝트는 AI 에이전트의 대화 흐름을
실제 사용 화면처럼 시뮬레이션하는 프론트엔드 UI 데모입니다.

--------------------------------------------------

[주요 기능]
- 채팅 UI 시뮬레이션
- 타이핑 인디케이터
- 자동 커서 이동 데모
- PDF 요약 카드
- 이메일 전송 카드
- 그래프 / 테이블 시각화
- CSV 다운로드

--------------------------------------------------

[기술 스택]
- HTML / CSS / Vanilla JavaScript
- 외부 라이브러리 미사용

--------------------------------------------------

[파일 구조]

index.html  : 메인 HTML
css.css     : 스타일 시트
js.js       : 채팅 로직 및 시뮬레이션 로직
README.txt  : 파일 설명

--------------------------------------------------

[실행 방법]

1. index.html 파일을 더블 클릭합니다.
2. 상단 [다시 재생] 버튼으로 데모를 재실행할 수 있습니다.

※ js.js는 defer 옵션으로 로드되어야 정상 동작합니다.

--------------------------------------------------

[데모 시나리오]

1. PDF 파일 요약 요청
2. 요약 내용 이메일 전달
3. 부서별 매출 그래프 생성
4. 설비 가동률 조회 및 CSV 다운로드

--------------------------------------------------

[주의 사항]

- 실제 AI API와 연동되어 있지 않습니다.
- 모든 데이터는 데모용 더미 데이터입니다.
- 브라우저 줌이 100%가 아닐 경우
  텍스트가 흐려 보일 수 있습니다.

--------------------------------------------------

[활용 목적]

- AI 에이전트 UX/UI 컨셉 데모
- B2B 서비스 제안용 프로토타입
